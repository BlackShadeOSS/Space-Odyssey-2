About The game

Space Odyssey 2 is a game created by Tomasz Tarnawski. It is a sequel to the game Space Odyssey. The game is a 2D platformer with a retro style. The game is about a space traveller who travels through space and fights with ufo. The game is created in JavaScript using the canvas element. The game is available on the website, small screens are not supported. It don't need to be downloaded and it is free.
